# 🎬 Ultra Cinematic 4K Intro Generator

High-end cinematic intro generator built with Python, OpenCV, MoviePy, and advanced visual effects.

Transforms a single image into a Hollywood-style animated intro with:

* Cinematic camera movement
* Dynamic lighting flashes
* Smoke & fire particle systems
* Motion blur & depth of field
* Golden cinematic color grading
* Glow effects & vignette
* 4K Ultra HD rendering
* Audio integration support

Designed for creators, editors, cinematic intros, gaming edits, trailers, and visual experiments.

## ⚡ Features

* 4K / 60FPS rendering
* Fully procedural visual effects
* Real-time cinematic transformations
* FFmpeg H264 export
* GitHub Actions compatible
* Highly customizable effect pipeline

## 🛠 Built With

* Python
* MoviePy
* OpenCV
* NumPy
* Pillow
* FFmpeg

## 📦 Output

Generates a cinematic MP4 intro video from a single storyboard image.

Hollywood atmosphere directly from Python.
